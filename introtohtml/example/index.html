Paragraph text. HTML uses tags, which are these things in angle brackets. It tells the computer what type of information you're giving it (text, images, etc). EVERYTHING goes inside the body tag.

For text, you can also put in a link, or do bold or italicized text, or even a customizable mystery third option called span.

A div is a box. I put some inline css to give this box a border, so you can see it.

(Fun fact: you can put a link tag around a div to make a clickable button!)
